# xmlify

## Getting Started

Add the xmlify dependency to your project

### Gradle

```kotlin
implementation("com.github.leomartins1999:xmlify:2.0.0")
```

### Maven

```xml
<dependency>
    <groupId>com.github.leomartins1999</groupId>
    <artifactId>xmlify</artifactId>
    <version>2.0.0</version>
</dependency>
```
